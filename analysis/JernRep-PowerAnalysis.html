Power Analysis: Jern et al. (2014) Replication

Author

Victor Moreno

Overview

Jern, Chang, and Kemp (2014) used a 3 (condition: polarization, moderation, control) x 2 (forced-choice groups: yes, no) between-subjects design. The primary outcome was whether participants became less certain, remained equally certain, or became more certain about their initial disease-class judgment after being presented with two contradictory test results. The primary comparisons of interest were polarization vs. control and moderation vs. control.

In this notebook, I recover the proportions for the polarization and moderation conditions from the reported z-statistics and sample sizes, calculate Cohen’s h for each comparison, and estimate the sample size required to detect these effects with 80% power (Cohen 1969). Although the original study used unequal sample sizes across cells, the present replication will allocate participants equally across the six experimental cells. Total N is therefore calculated by multiplying the required sample size per cell by 6 (3 conditions x 2 forced-choice groups). Effect sizes are estimated from the no-forced-choice group because the reported effects were smaller than those in the forced-choice group, yielding a more conservative sample size estimate. All analyses are conducted in R (R Core Team 2025), following the approaches outlined in Frank et al. (2025).

Recover unreported proportions

The authors reported the proportions of participants in the control condition who became less certain, remained equally certain, and became more certain. However, to estimate the sample size required to detect these effects with 80% power, we also need the corresponding outcome proportions for the polarization and moderation conditions, which were not reported.

To recover these values, we solve the equation for a one-tailed z-test for proportions, treating the reported control proportion as p2 and solving for the value of p1 (the polarization or moderation proportion) that reproduces the reported z-statistic: \[ z = \frac{p_1 - p_2} {\sqrt{\hat{p}(1-\hat{p}) \left(\frac{1}{n_1}+\frac{1}{n_2}\right)}} \]

where

\[ \hat{p}= \frac{p_1n_1+p_2n_2}{n_1+n_2}. \]

## Recover p1
calc_p1 <- function(z_reported, p2, n1, n2) {
  eq <- function(p1){
    ## proportion pooled across conditions 
    p_pool <- (p1 * n1 + p2 * n2) / (n1 + n2)
    ## standard error of the difference in proportions under the null
    se <- sqrt(p_pool * (1 - p_pool) * (1/n1 + 1/n2))
    ## difference between the z-statistic implied by p1 and the reported z-statistic
    ((p1-p2) / se) - z_reported
  }
  ## solve for p1 s.t the difference is 0 
  result <- uniroot(eq, 
                    interval = c(p2, 0.99))
  ## output
  return(result$root)
}

## Polarization proportion (more certain)
p_pol_more <- calc_p1(
  z_reported = z_pol,
  p2 = p_ctrl_more, ## control 
  n1 = n_pol, ## polarization
  n2 = n_ctrl ## control
)

## Moderation proportion (less certain)
p_mod_less <- calc_p1(
  z_reported = z_mod,
  p2 = p_ctrl_less, ## control 
  n1 = n_mod, ## moderation
  n2 = n_ctrl ## control
)

## Output recovered proportions

Calculate Cohen’s h

Next I use p1 (the polarization or moderation proportion) and p2 (control) to calculate Cohen’s h (Cohen 1988):

\[ h = 2\arcsin\left(\sqrt{p_1}\right) - 2\arcsin\left(\sqrt{p_2}\right) \] Effect sizes are computed using the pwr package (Champely 2020).

library(pwr)
## Cohen's h for polarization vs control
h_pol <- ES.h(p_pol_more, p_ctrl_more)
cat("Cohen's h for polarization vs control: ", 
    round(h_pol, 3), 
    "\n")
Cohen's h for polarization vs control:  0.334 
## Cohen's h for moderation vs control  
h_mod <- ES.h(p_mod_less, p_ctrl_less)
cat("Cohen's h for moderation vs control: ", 
    round(h_mod, 3), 
    "\n")
Cohen's h for moderation vs control:  0.284

Calculate n required per experimental cell

Next, I calculate the n required per cell to detect the reported effect sizes at 80% power:

## n for polarization vs control
power_pol <- pwr.2p.test(h = h_pol, 
            sig.level = .05, 
            power = .80, 
            alternative = "greater")
cat("n per cell required (polarization vs control): ", 
    ceiling(power_pol$n), 
    "\n")
n per cell required (polarization vs control):  111 
## n for moderation vs control 
power_mod <- pwr.2p.test(h = h_mod,

            power = .80,
